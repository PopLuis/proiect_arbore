## ** Pasii pentru utilizare**

1. **Descarcare imagine:**
```bash
docker pull ghcr.io/popluis/proiect_arbore/proiect_map:latest
```

2. **Rularea imaginii:**
```bash
docker run ghcr.io/popluis/proiect_arbore/proiect_map:latest
```
